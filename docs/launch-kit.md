## Short Intro

English:

`Svate is an open-source Android Claw. It can read the UI tree, capture the screen, upload frames through signed URLs, and execute actions on-device through Accessibility. The backend handles planning, validation, telemetry, and safety-sensitive decision support.`

Chinese:

`Svate 是一个开源的 Android Claw。它可以读取 UI 树、截取屏幕、通过签名 URL 上传截图，并通过 Accessibility 在设备上执行动作。后端负责规划、校验、遥测和安全相关的决策支持。`

## What To Emphasize

- It is an Android Claw, not just another chat wrapper.
- It combines UI tree access and visual perception instead of relying on screenshots alone.
- It has explicit safety and verification layers.
- It includes both Android runtime code and the backend needed to support agent-style execution.
